Archived Data

Some users have requested that we post some past data. This data may be useful as a comparison for what you found in your class, or it could be substituted for your own data in case you were unable to run the experiment but still wanted the students to do the analysis.

If you would like some of your own data posted, either send the URL of an appropriate link or an ascii file in a format similar to the data posted here to to miller@zia.hss.cmu.edu.

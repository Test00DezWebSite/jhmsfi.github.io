WHAT IS IT?

This model demonstrates the diffusion of a joke in a randomly created spatial network.


HOW IT WORKS

In each tick every 'laughing' (having just heard the joke) node spreads it to its neighbors if it considers the joke funny, short and appropriate enough to pass it on.


HOW TO USE IT

Choose the size of randomly created spatial network using the 'Number-of-nodes' slider.
Choose the average density of links in the network using the 'Average-node-degree' slider.

To create the network with these properties, press SETUP.

With the 'Funniness' slider, the 'Words' slider, and the 'Appropriateness' slider the quality of the joke can be changed. The higher the value the better: thus the funnier, the shorter, and the more appropriate the joke is.

With the 'Laughing-threshold' the lower limit can be chosen of how good quality the joke at least has to be that an agent likes to pass it on.

Press the GO button to run the model.


THINGS TO NOTICE

The quality of the joke, the threshold of each agent's humor as well as the topology of the social network have strong impacts on the persistence and spread of a joke, can prevent it from being spread or increase the speed of its diffusion.


PROCEDURES

turtles-own 
[ val new-val  
  laughing?
]


to setup
  clear-all
  setup-nodes
  setup-spatially-clustered-network
  ask one-of turtles 
  [ set laughing? true 
    set color red    
  ]
end

to setup-nodes
  set-default-shape turtles "circle"
  crt number-of-nodes
  [
    setxy (random-xcor * 0.95) (random-ycor * 0.95)
    set color blue
    set laughing? false
  ]
end

to setup-spatially-clustered-network
  let num-links (Average-node-degree * Number-of-nodes) / 2
  while [count links < num-links ]
  [
    ask one-of turtles
    [
      let choice (min-one-of (other turtles with [not link-neighbor? myself])
                   [distance myself])
      if choice != nobody [ create-link-with choice ]
    ]
  ]
  repeat 10
  [
    layout-spring turtles links 0.3 (world-width / (sqrt Number-of-nodes)) 1
  ]
end


to go
if all? turtles [laughing?]
[stop]
 spread-joke
 tick   
end  
  
to  spread-joke
  ask turtles with [laughing?]
  [ask link-neighbors with [Funniness + Words + Appropriateness + 1 > Laughing-threshold ] 
    [set laughing? true
      set color red
    ]
  ]
end
